GeoConstructor - объектно ориентированный программный интерфейс для создания приложений для геоинформационных систем в среде MS Windows, разработанный по стандарту Visual Basic Custom Control (VBX) и ActiveX (OCX). GeoConstructor поставляется вместе с GeoGraph и GeoDraw, и предназначен для широкого круга пользователей ГИС, которые хотели бы самостоятельно заняться созданием своего собственного прикладного ПО для ГИС.

Основные возможности GeoConstructor v. 1.5

  • Большая свобода в выборе среды разработки. Протоколы VBX и OCX поддерживает большинство современных средств визуальной разработтки приложений - Microsoft Visual Basic, Visual C++, Borland C++, Borland Delphi и др.
  • Возможность создания ГИС-приложений, работающих с широким спектром форматов данных. GeoConstructor будет поддерживать работу с любыми цифровыми картами в формате ГИС GeoDraw/GeoGraph, а с учетом имеющейся в этой ГИС возможностей импорта - c картами из широкого спектра исходных форматов - ARC/INFO, MapInfo, DXF, F1M, DX-90 и др.
  • Используя поставляемые с пакетами Microsoft Visual Basic, Visual C++ или с Delphi драйверы для доступа к базам данных ODBC или IDAPI, можно создать ГИС-приложения, работающие с форматами dBase, Paradox, Access, Oracle, Sybase, Informix, InterBase и др.
  • Возможность создания тематических карт.
  • Возможность импорта картографических композиций, созданных в системе GeoGraph.
  • Поддержка редактирования пространственных данных (создание новых и редактирование существующих нетопологических (косметических) слоев).
  • Возможность редактирования отдельных вершин полигонов и дуг, модификация границ объектов на основе подобия фигур, а также стандартные операции с буфером обмена Windows.
  • Средства программного управления композицией карты в создаваемом ГИС приложении.
  • Наличие встроенных инструментов управления изображением карты и доступа к объектам, включая координаты объектов.
  • Широкий спектр возможностей для произвольной и условной выборки и фильтрации пространственных объектов.
  • Разнообразные способы управления масштабом изображения.
  • Поддержка географических проекций.
  • Поддержка растровых слоев большинства современных форматов (BMP, TIFF, GIF, PCX и др.).
  • Простота и доступность изучения и использования (для обучения программированию с использованием GeoConstructor лиц, владеющих основами указанных выше языков программирования, достаточно нескольких дней).

Основные возможности * Новшества в версии 1.51

О версии 1.5

В GeoConstructor версии 1.5 введены многие новшества, появившиеся в соответствующей версии ГИС GeoGraph. К сожалению, полной совместимости с предыдущей версией добиться не удалось. Однако, переделка уже существующих приложений, основанных на GeoConstructor 1.1, потребует минимальных усилий со стороны разработчиков, так как все фундаментальные принципы программирования остались неизменными. В основном придется модифицировать код, относящийся к присвоению объектам графических атрибутов.

Новые возможности GeoConstructor v.1.5:

Новая модель графических атрибутов картографических объектов
Теперь модель раскраски объектов основана на использовании подключаемых библиотек шаблонов атрибутов. Эти библиотеки Вы можете создавать и редактировать в среде GeoGraph 1.5, а также с помощью отдельно поставляемого редактора.

Новые форматы слоев
Помимо слоев GeoDraw для DOS, с которыми GeoConstructor работал ранее, теперь Вы можете работать со слоями GeoDraw для Windows, а также с косметическими слоями GeoGraph 1.5. Все сторонние форматы, поддерживаемые GeoGraph 1.5 (SXF, DX90), могут использоваться только через импорт карт GeoGraph 1.5.

Встроенный векторный редактор
Реализованы все функции векторного не топологического редактора для косметических слоев GeoGraph 1.5. Вы можете создавать и редактировать эти слои как программно, так и с помощью встроенных инструментов мыши.


Импорт карт, созданных в среде GeoGraph 1.5
ГеоКонструктор может импортировать как старые карты, созданные в GeoGraph 1.1, так и новые.


Динамический растровый оверлей площадей
Теперь в GeoConstructor полностью реализован растровый оверлей, который был уже в версии 1.1 ГИС GeoGraph. Вы получаете довольно мощную аналитическую функцию, которая позволит отображать и анализировать сложные запросы с разнородными условиями.

Новшества в версии 1.51

Версия 1.51 полностью обратно совместима с версией 1.5. Программы написанные на Delphi необходимо перекомпилировать. Это обусловлено тем, что вызовы свойств в Delphi реализованы по индексам, а не по названиям.

Расширены функции встроенного графического редактора. Помимо создания, удаления и перемещения объектов теперь появилась возможность редактирования отдельных вершин полигонов и дуг, модификация границ объектов на основе подобия фигур, а также стандартные операции с буфером обмена Windows.

В редактор также встроена теперь возможность одношаговой операции UNDO/REDO.

Устранены некоторые нестыковки с Windows NT. В основном они касались работы Win32-программ, задействующих возможности пользовательского рисования на карте. Однако, полностью избавиться от влияния NT не удалось из-за несовместимости графических контекстов 32-разрядных и 16-разрядных программ. Эти проблемы будут сняты в последующей полностью 32-разрядной версии GeoConstructor.

Включен 32-разрядный ActiveX компонент-оболочка (GC151.OCX) предназначенный для возможности использования GeoConstructor в любой Win32 среде программирования (Microsoft Visual Basic 4/5, Microsoft Visual C++ 4/5, Borland Paradox 8.0 и т.п.).
