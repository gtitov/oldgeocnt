GeoGraph - end-user level GIS.

GeoGraph is intended for end-users, who mainly work with existing digital maps, rasters and related databases.

GeoGraph is used during 1992-1998 on thousands maps and plans from 1:500 scale to 1:50 000 000 by users in forest, geology, land, transport, geodesy and cartography, military, election, education and research fields etc. It really takes only a few days for training before working.

GeoGraph works under Windows 3.1/95/98/NT 3.51/NT 4.0/2000/XP on Intel platform.

GeoGraph is one of the most popular, professional, inexpensive and easy-to-use end-user GIS in Russia. The price of 1 installation is US$500, including 1 year support and free new versions.

It offers the following functionality:

  • Digital map control
    Combining thematic electronic atlases and maps as compositions of digital map layers. Pan and zoom map.
  • Thematic mapping
    Displaying data using multiple classifications, graduated symbols, shade intensity and labeling.
  • Multiple spatial data formats
    It is based on a set of drivers for access to vector and raster layers of various formats. It performs direct reading of spatial data from an open set of different formats - - GeoDraw, DX-90, russian topographic service formats (SXF, F23), ArcView shape file, about 40 raster formats (TIFF, GIFF, BMP, PCX,JPEG, SPOT IMAGE etc.), special multi-map-sheet layer for any format etc.
  • Cosmetic layers management
    Fast and easy drawing and editing of graphical objects such as point, line, polygon on map and storing them in multiple cosmetic layers.
  • User styles for spatial objects
    Providing multiple symbol libraries for points, lines and polygons drawing. Patterns, line styles and symbol editing.
  • Map and database connection
    Direct access to data stored in different popular database formats:
    Paradox, dBase, Foxpro, MS Access
    "Client-Server" connection with different SQL Servers:
    MS SQL Server, Oracle, Interbase, DB2 ect.
  • Various queries and retrievals
    Query By Example language support, storing multiple queries for any layer data, calculating of new field values using expression builder, performing powerfull queries using SQL.
  • Screen form designer
    You can construct your own screen forms and reports for a better view of attributive data.
  • Macro commands
    You can link spatial object or screen form element with macro commands which invoke defined action such as run external program, load map composition, play video or sound tracks, map magnification, proceed query and others.
  • Dynamic Data Exchange
    GeoGraph works as DDE server to perform commands and exchange data with other applications.
  • Measuring tools
    Mesuring of areas, distances, perimeters according scale, unit settings and map projection.
  • Print layout control
    Print layout designer for map, reports, charts, legend output. Printing of big map compositions on smaller size equipment with automatic splitting on the sheets.
  • Spatial data analysis
    "On fly" dynamic raster overlay of layers to find critical or suitable zones by selection of objects from different layers, buffer zones calculations, spatial selections for ex., find all communications crossing the selected land parcels etc
