ГИС ГеоГраф 1.1
Программный продукт предназначен для конечных пользователей ГИС.
Версия создана в 1994 г. для среды Windows 3.1 и распространялась в 1995-1996 гг.
Незащищенная версия распространяется без технической поддержки.
Содержит ограниченный функционал.

в обменный формат MapInfo MIF/MID. 100 если на машине установлен GC 2.0.100, 93-97 для версий GC 2.0.93.-2.0.97

- BDE 32-х разрядное;
