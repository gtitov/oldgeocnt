Дополнительные приложения для ГИС ГеоГраф

 

14/07/98 Центр геоинформационных исследований завершает разработку новых дополнительных приложений для ГИС ГеоГраф.

24/12/98 Выпущена новая версия программы для создания диаграмных условных знаков (картодиаграмм). Добавлены следующие функции:
- Возможность задания штриховок для элементов картодиаграмм

26/11/98 Обращаем Ваше внимание! Центром Геоинформационных Исследований выпущены полезные программы для построения буферных зон и диаграмных условных знаков (картодиаграмм). Обе программы работают под управлением системы ГеоГраф. Более подробную информацию Вы можете узнать из файлов справки, поставляемых с программами.
Обращаем Ваше внимание на то, что приложение для построения буферных зон работает только при установленном на компьютере ПО ГеоКонструктор 1.60 !!!

23/10/98 Обращаем Ваше внимание! Центром Геоинформационных Исследований выпущен Редактор файлов описания проекции. Он будет полезен тем, кто имеет дело с преобразованиями географических проекций карт в системе GeoDraw/GeoGraph, поскольку позволяет существенно упростить процесс создания файлов описания проекции, используемых для преобразований. Более подробную информацию Вы можете подчерпнуть из файла справки, поставляемого с программой.
