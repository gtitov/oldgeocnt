GeoGraph GIS 2.0

GeoGraph is the multi-functional GIS tool :

- this is the professional and easy-to-use tool for creation of high quality digital maps using different ways and sources (digitizer, raster images from more than 30 different formats, field survey data, coordinate descriptions as texts and databases tables, formats of other types of GIS/CAD products etc.);

  • full topological editing compatible with ARC/INFO (proved on hundreds work places having permanent exchange with ARC/INFO since 1992) as well as non-topological editing is available in the same product GeoGraph 2.0;
  • all required map projections and coordinate systems are supported as well as a wide set of planar transformation by control points - affine, projective, high precision local-affine transformation; the transformations are available both for vector and raster data, for creation of multi-sheet structure etc.;
  • easy data exchange with other GIS; export to/import from ARC/INFO/ArcView, MapInfo, AutoCAD, GeoMedia, Microstation etc., conversion of many layers/sheets/tables by one step; drivers for straight working in GeoGraph with the data of ArcView (shapefiles/tables, raster geo-referenced world files), MapInfo, geo-referenced raster space images and more than 30 raster formats; potentially drivers for any format can be added without changes inside the system; working with DBF, DB tables formats and with any RDBMS via BDE/ODBC drivers ( MS Access, Oracle, Interbase, MS SQL server etc.);
  • wide set of "map quality control and correction" functions

- wide set functionalities of end-user and advanced-user GIS functionalities:

  • multi-document interface, context-oriented environment
  • projects/map compositions can be built from vector/raster layers of different formats/projections and attribute data tables of different formats;
  • enhanced thematic mapping functions, free cartographic symbol editor (point, line, polygon cartographic signs, wide set of libraries), enhanced label (annotation) location functions
  • presentation of maps in any required map projections "on fly"
  • searching on map/tables, data filters, selection and queries functions, QBE, SQL, form designer, macros designer,
  • GIS analytical functions : overlay "on fly", buffer zones, spatial/database selections through layers, dynamic segmentation
  • OLE automation
  • enhanced layout for printing to any devices (including many maps, rasters, forms, tables, texts etc.)
  • export to large-size BMP and WMF files for professional publishing systems like Adobe Photoshop/Illustrator etc.
  • version of GeoGraph-mini for free (for administrators, officials etc.)
