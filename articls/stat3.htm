Программное обеспечение для конечного пользователя. ГеоГраф.

      ГеоГраф дает возможность пользователям реализовать большую часть из основного набора функций ГИС уровня конечного пользователя, имеющихся в наиболее известных зарубежных программных продуктах, и ряд дополнительных функций.  

          В частности, ГеоГраф позволяет :

  • создавать электронные карты и атласы как композиции слоев цифровых карт, связанных с ними таблиц атрибутивных данных, растровых изображений. Система построена по принципу драйверов форматов, что позволяет напрямую работать с любыми векторными картами и растровыми изображениями, форматы которых открыты и описаны, а также с любыми базами атрибутивных данных, для которых имеются драйверы ODBC или IDAPI;

  • строить электронные тематические карты по любым слоям и табличным данным путем классификации данных и отображения объектов любого класса с помощью графических переменных (цвета и интенсивность заливок и штриховок, толщины и формы линий, размер, форма и ориентация точечных знаков), выбранных или созданных пользователем;

  • создавать нетопологические слои пространственных объектов (например, оперативной обстановки) непосредственно в системе конечного пользователя в виде, пригодном для передачи по сети, радиоканалу и прочим коммуникациям, редактировать их в режиме многопользовательского доступа;

  • строить разнообразные запросы, выборки как по табличной, так и по пространственной информации или по их сочетанию, привязывать к слоям или объектам разнообразные макросы, включая загрузку пользовательских программ, новых композиций карт, цифровой аудио и видеоинформации, связанной с объектами карт, редактировать таблицы, проводить в них вычисления, строить и проигрывать QBE и SQL- запросы;

  • проводить измерения по карте, в том числе и с записью результатов в базу (площади, расстояния, координаты объектов);

  • выполнять динамический оверлей слоев (благоприятные или неблагоприятные зоны по комплексу слоев, построение в реальном времени в зависимости накладываемых условий и выбора слоев). Выполнять пространственные селекции (например, все коммуникации, проходящие через выбранные участки);

  • реализовывать диспетчерские функции (регенерация тематических карт по отслеживаемым изменениям в базе атрибутивных данных и др.);

  • формировать композицию карты для вывода на печать, включая любые фрагменты карты, тексты, растровые изображения, графики, легенду, формы справок, таблицы. Готовить цифровые карты для передачи в издательские системы;

      ГеоГраф позволяет включать в себя приложения, реализующие различные расчетно-аналитические и прикладные задачи, использующие интерфейс ГеоГраф и данные, отвечающие требованиям ГеоГраф (для их подготовки не нужны какие-то иные средства). В качестве примеров таковых можно назвать :

      ГеоФит - комплекс алгоритмов и программ интерполяции на регулярную сеть и фильтрации случайных компонент самых различных характеристик, измеренных на нерегулярной сети наблюдений, универсальные относительно видов съемок, этапов их проведения, детальности и степени неоднородности исходной сети данных. Комплекс может быть использован при обработке геологических, геохимических, геофизических, экологических, географических и других видов наблюдений для получения значений по регулярной сети данных в целях построения карт, проведения тренд-анализа и дальнейших расчетов по усмотрению пользователя. В частности ГеоФит может быть использован для построения изолиний по результатам точечных измерений.

 

      ГеоБуф - приложение для построения буферных зон вокруг точечных, линейных и площадных объектов.

      ГеоМаршрут - приложение для расчета оптимальных путей движения по сети. Можно использовать для решения задачи оптимизации пути следования транспортных средств.

      ГеоДин - приложение для решения задачи динамической сегментации - размещения точечных и линейных объектов вдоль линейных трасс - автодорог, инженерных коммуникаций и других линейных объектов, на основе фиксации расстояния от начальных точек участков трасс.


      Подчеркиваем, что для реализации указанных приложений не требуются какие-то иные специальные и дорогостоящие программные средства подготовки для них данных.
